## 🏛️ 功能展示

### 1️⃣ 智囊团 — 三专家独立投票

> 技术派看趋势 + 价值派看估值 + 资金派看主力动向 → 三票综合

```text
🏛️  贵州茅台(600519) → 🟢 推荐买入 (评分:+1.4)
───────────────────────────────────────
 ⚪ 技术派 → 观望(+0)  今日窄幅震荡
 🟢 价值派 → 买入(+1)  PE 20.1 估值合理
 🟢 资金派 → 买入(+3)  大宗交易 ¥9123 万 · 主力扫货
───────────────────────────────────────
 🗳️  投票结果 —— 买入 2 票 · 卖出 0 票

 💡 建议：三专家中两位看多，可逢低建仓
```

```text
🏛️  五粮液(000858) → ⚪ 观望 (评分:+0.7)
───────────────────────────────────────
 ⚪ 技术派 → 观望(+0)  今日窄幅震荡
 🔴 价值派 → 卖出(-1)  PE 26.4 估值偏高
 🟢 资金派 → 买入(+3)  大宗交易 ¥42327 万
───────────────────────────────────────
 🗳️  投票结果 —— 买入 1 票 · 卖出 1 票 · 专家分歧

 💡 建议：资金面偏好但估值偏高，观望为主
```
